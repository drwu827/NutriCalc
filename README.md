## 指標說明

### FFMI（Fat-Free Mass Index）

FFMI 為去脂體重指數，可協助評估肌肉量相對於身高的表現，比單純 BMI 更適合有運動習慣者。

### WHtR（Waist-to-Height Ratio）

WHtR 為腰高比，可協助評估腹部脂肪風險。

### LBM（Lean Body Mass）

LBM 為去脂體重，代表身體扣除脂肪後的重量。

### TDEE（Total Daily Energy Expenditure）

TDEE 為每日總消耗熱量，是飲食規劃的重要基礎。
